# Corp-Fin-Proj

## Berry Global Financial Analysis & Valuation

This project contains a comprehensive corporate finance analysis of Berry Global Group, Inc. (NYSE: BERY), including:

- Financial Statement Analysis
- Discounted Cash Flow (DCF) Valuation
- Investment Thesis
- Risk Analysis
- Industry & Peer Comparison
- Growth Drivers
- Strategic Merger Analysis (Amcor Deal)

---

## Files Included

| File | Description |
|------|-------------|
| `Berry Global.xlsx` | Financial analysis and ratio calculations |
| `Berry Global DCF.xlsx` | DCF valuation model |
| `Berry_Global_Portfolio_Report.docx` | Full equity research report |

---

## Key Highlights

- Revenue growth from $11.7B (2020) to estimated $13.3B (2024E)
- Strong Free Cash Flow generation (~$800M annually)
- Undervalued compared to peers on P/E and EV/EBITDA basis
- Sustainability-focused packaging innovation strategy
- Strategic merger with Amcor expected to unlock major synergies

---

## Investment Recommendation

**Rating: BUY**

Berry Global presents a compelling value opportunity supported by:

- Stable cash flows
- Strong operational execution
- Global market presence
- Sustainability initiatives
- Attractive valuation discount
